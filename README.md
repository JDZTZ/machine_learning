## 代码运行

### 人工智胀 ｜ 堂妹教你用KNN算法筛选约会对象
首先进入文件夹
```
cd knn
```
运行KNN代码
```
python test_knn.py
```
数据归一化后运行KNN
```
python test_knn.py --autonorm  
```
网格法搜索求最佳特征置信度
```
python test_knn.py --autonorm --grid_search
```
